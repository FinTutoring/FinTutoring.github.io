1. Valeur Temporelle de l'Argent (TVM)

Intuition Fondamentale

Un euro aujourd'hui vaut plus qu'un euro demain car il peut être investi pour générer des revenus. C'est le principe de base de toute la finance !

Formule de Base - Valeur Future

FV = PV × (1 + r)ⁿ

FV = Valeur Future

PV = Valeur Présente

r = Taux d'intérêt par période

n = Nombre de périodes

Valeur Actuelle Nette (VAN)

VAN = Σ [CFₜ / (1 + r)ⁿ] - Investissement Initial

CFₜ = Flux de trésorerie à la période t

r = Taux d'actualisation

Exemple Pratique

Situation : Vous investissez 10 000€ à 5% pendant 3 ans.

Calcul : FV = 10 000 × (1 + 0,05)³ = 10 000 × 1,157625 = 11 576,25€

Interprétation : Votre investissement vaudra 11 576,25€ dans 3 ans.

Applications CFA

  • Évaluation d'obligations : Calcul du prix basé sur les flux futurs
  • Analyse d'investissement : Comparaison de projets avec VAN et TRI
  • Planification financière : Objectifs de retraite et d'épargne
  • Valorisation d'actions : Modèles de dividendes actualisés

2. Statistiques Descriptives et Probabilités

Intuition

Les statistiques nous permettent de comprendre et quantifier le comportement des marchés financiers, notamment le risque et le rendement.

Moyenne Arithmétique

x̄ = Σxᵢ / n

Rendement moyen d'un actif

Écart-Type (Volatilité)

σ = √[Σ(xᵢ - x̄)² / n]

Mesure du risque

Coefficient de Variation

CV = σ / μ

Risque relatif au rendement

Exemple : Analyse d'un Portefeuille

Rendements mensuels d'une action :

Jan: +2%, Fév: -1%, Mar: +3%, Avr: +1%, Mai: -2%

Calculs :

Moyenne = (2-1+3+1-2)/5 = 0,6%

Écart-type = 1,95%

CV = 1,95/0,6 = 3,25

3. Distributions Statistiques

Intuition

Les distributions décrivent comment les rendements se répartissent. La distribution normale est fondamentale en finance car de nombreux phénomènes financiers la suivent approximativement.

Distribution Normale

  • 68% des valeurs dans μ ± 1σ
  • 95% des valeurs dans μ ± 2σ
  • 99,7% des valeurs dans μ ± 3σ
  • Symétrique autour de la moyenne
  • Asymptote aux extrêmes

Distribution t-Student

  • Utilisée pour petits échantillons (n < 30)
  • Plus large que la normale
  • Converge vers normale quand n → ∞
  • Degrés de liberté = n - 1
  • Essentielle pour les tests d'hypothèses

Calcul de Probabilité - Loi Normale

Z = (X - μ) / σ

Exemple : Si les rendements d'une action suivent N(8%, 15%), quelle est la probabilité d'avoir un rendement > 20% ?

Solution : Z = (20-8)/15 = 0,8 → P(Z > 0,8) = 0,2119 = 21,19%

4. Théorie Moderne du Portefeuille

Intuition de Markowitz

La diversification permet de réduire le risque sans sacrifier le rendement. "Ne mettez pas tous vos œufs dans le même panier" - principe mathématiquement prouvé !

Rendement du Portefeuille

E(Rₚ) = Σ wᵢ × E(Rᵢ)

Moyenne pondérée des rendements

Risque du Portefeuille

σₚ² = Σ wᵢ²σᵢ² + 2Σ Σ wᵢwⱼσᵢⱼ

Prend en compte les corrélations

Effet de la Diversification

Corrélation = +1

Aucun bénéfice de diversification

Corrélation = 0

Diversification modérée

Corrélation = -1

Diversification parfaite

Exemple : Portefeuille 2 Actifs

Données :

Action A: E(R) = 10%, σ = 20%

Action B: E(R) = 15%, σ = 30%

Corrélation = 0,3

Poids : 60% A, 40% B

Résultats :

E(Rₚ) = 0,6×10% + 0,4×15% = 12%

σₚ = √[(0,6²×20²) + (0,4²×30²) + 2×0,6×0,4×20×30×0,3]

σₚ = 19,2% (< 22,8% sans diversification)

5. Tests d'Hypothèses - TERMINOLOGIE CORRIGÉE

Section avec corrections importantes sur la terminologie statistique

ERREUR CRITIQUE À ÉVITER

❌ INCORRECT

  • • "Accepter l'hypothèse nulle"
  • • "Accepter l'hypothèse alternative"
  • • "L'hypothèse est prouvée"
  • • "L'hypothèse est vraie"

✅ CORRECT

  • • "Rejeter l'hypothèse nulle"
  • • "Ne pas être en mesure de rejeter H₀"
  • • "Les données ne permettent pas de rejeter H₀"
  • • "Insuffisant pour rejeter H₀"

Pourquoi cette distinction est CRUCIALE

Philosophie statistique : Un test d'hypothèse ne peut jamais "prouver" qu'une hypothèse est vraie, seulement qu'elle est compatible ou incompatible avec les données observées.

Analogie juridique : Un tribunal déclare "coupable" ou "non coupable", jamais "innocent". De même, on "rejette H₀" ou "ne rejette pas H₀".

Implications pratiques : "Ne pas rejeter H₀" ≠ "H₀ est vraie". Cela signifie simplement que les preuves sont insuffisantes.

Processus de Test

  1. Formuler H₀ et H₁
  2. Choisir le niveau de signification α
  3. Calculer la statistique de test
  4. Déterminer la valeur critique
  5. Conclusion : Rejeter ou ne pas rejeter H₀

Test t pour une Moyenne

t = (x̄ - μ₀) / (s / √n)

Degrés de liberté = n - 1

Exemple avec Terminologie Correcte

Contexte : Test si le rendement moyen d'un fonds est différent de 8%

H₀ : μ = 8% vs H₁ : μ ≠ 8%

Données : n = 25, x̄ = 9,2%, s = 4%, α = 5%

Calculs :

t = (9,2 - 8) / (4 / √25) = 1,5

Valeur critique (bilatéral) = ±2,064

|1,5| < 2,064

Conclusion CORRECTE :

✅ "Nous ne sommes pas en mesure de rejeter H₀ au seuil de 5%"

❌ PAS : "Nous acceptons que μ = 8%"

Conseils pour l'Examen CFA

  • Vocabulaire précis : Utilisez toujours "rejeter" ou "ne pas rejeter", jamais "accepter"
  • Interprétation : "Ne pas rejeter H₀" signifie "preuves insuffisantes", pas "H₀ est vraie"
  • Erreurs de Type I et II : Liées aux fausses conclusions, pas aux "acceptations"
  • P-value : Probabilité d'observer des résultats au moins aussi extrêmes si H₀ était vraie

6. Régression et Corrélation

Intuition

La régression nous permet de modéliser et prédire les relations entre variables financières. C'est l'outil de base pour comprendre comment les facteurs influencent les rendements.

Équation de Régression

Y = α + βX + ε

α = Ordonnée à l'origine (alpha)

β = Pente (bêta)

ε = Terme d'erreur

Coefficient de Corrélation

r = Cov(X,Y) / (σₓ × σᵧ)

-1 ≤ r ≤ +1

r² = Coefficient de détermination

Interprétation du R²

R² = 0,25

25% de la variance expliquée

R² = 0,50

Relation modérée

R² = 0,75

Forte relation

R² = 0,90

Très forte relation

Application : Modèle CAPM

Équation CAPM :

Rᵢ - Rf = αᵢ + βᵢ(Rm - Rf) + εᵢ

Interprétation du Bêta :

  • • β = 1 : Risque identique au marché
  • • β > 1 : Plus risqué que le marché
  • • β < 1 : Moins risqué que le marché
  • • β < 0 : Corrélation négative

Test de l'Alpha :

H₀ : α = 0 (pas de sur-performance)

H₁ : α ≠ 0

✅ Si on rejette H₀ : sur-performance significative
