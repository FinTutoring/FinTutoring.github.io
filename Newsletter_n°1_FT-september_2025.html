Advanced Formula Analysis

Black-Scholes Model with Dividend Yield
European Call Option Pricing with Continuous Dividend Yield
C = S₀e⁻ᵍᵀN(d₁) - Ke⁻ʳᵀN(d₂)
Where:
d₁ = [ln(S₀/K) + (r - q + σ²/2)T] / (σ√T)
d₂ = d₁ - σ√T
Variables:
S₀ = Current stock price
K = Strike price
r = Risk-free rate
q = Dividend yield
σ = Volatility
T = Time to expiration

Model Applications & Extensions

Foreign Exchange Options: Garman-Kohlhagen model for currency pairs
Index Options: Incorporating dividend yields for equity index derivatives
Commodity Options: Convenience yield adjustments for storage costs
The dividend-adjusted Black-Scholes model serves as the foundation for more complex derivatives pricing, including barrier options, Asian options, and structured products.

Greeks Analysis Framework

Delta (Δ): ∂C/∂S = e⁻ᵍᵀN(d₁) - Price sensitivity to underlying movement
Gamma (Γ): ∂²C/∂S² = e⁻ᵍᵀφ(d₁)/(S₀σ√T) - Delta sensitivity
Theta (Θ): -∂C/∂T - Time decay acceleration
Vega (ν): ∂C/∂σ = S₀e⁻ᵍᵀφ(d₁)√T - Volatility sensitivity
Risk management teams use these Greeks to construct delta-neutral portfolios and hedge volatility exposure across options books.

Python Implementation Example

import numpy as np
from scipy.stats import norm

def black_scholes_dividend(S, K, T, r, q, sigma, option_type='call'):
    """
    Black-Scholes option pricing with continuous dividend yield
    """
    d1 = (np.log(S/K) + (r - q + 0.5*sigma**2)*T) / (sigma*np.sqrt(T))
    d2 = d1 - sigma*np.sqrt(T)
    
    if option_type == 'call':
        price = S*np.exp(-q*T)*norm.cdf(d1) - K*np.exp(-r*T)*norm.cdf(d2)
        delta = np.exp(-q*T)*norm.cdf(d1)
    else:  # put option
        price = K*np.exp(-r*T)*norm.cdf(-d2) - S*np.exp(-q*T)*norm.cdf(-d1)
        delta = -np.exp(-q*T)*norm.cdf(-d1)
    
    gamma = np.exp(-q*T)*norm.pdf(d1)/(S*sigma*np.sqrt(T))
    theta = (-S*norm.pdf(d1)*sigma*np.exp(-q*T)/(2*np.sqrt(T)) 
             - r*K*np.exp(-r*T)*norm.cdf(d2) 
             + q*S*np.exp(-q*T)*norm.cdf(d1))
    vega = S*np.exp(-q*T)*norm.pdf(d1)*np.sqrt(T)/100
    
    return {
        'price': price,
        'delta': delta,
        'gamma': gamma,
        'theta': theta/365,  # per day
        'vega': vega
    }

# Example usage
result = black_scholes_dividend(S=450, K=450, T=21/365, r=0.045, q=0.018, sigma=0.18)
print(f"Call Price: ${result['price']:.2f}")
print(f"Delta: {result['delta']:.3f}")
